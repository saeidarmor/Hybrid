# A Hybrid Approach for Efficient Similar Diagnosis Retrieval
Integrating Data Mining with High Performance Computing

## Description
This project aims to improve existing methods for similar diagnosis retrieval in Medical Information Systems (MIS). We implement and compare:
1. Data Normalization + Monte Carlo voting simulation
2. K-Means clustering
3. Parallel K-Means (Numba)
